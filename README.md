RESTful API for iReporter, an application for bringing corruption incidences to the notice of the general
public.

## RESTful API Endpoints Implemented
* GET /red-flags
* POST /red-flags
* GET /red-flags/<id>
* DELETE /red-flags/<id>
* PATCH /red-flags/<id>/location
* PATCH /red-flags/<id>/comment
